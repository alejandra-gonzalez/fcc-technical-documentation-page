Introduction

CSS is the mechanism that we use to describe the styling of web documents, such as those written in HTML. It is a style sheet language that allows you to pick HTML elements and customize their appearance.

p { color: purple }

The code above turns the text in a paragraph purple. This is one of the many things you can do with CSS. This guide will help you get started on working with CSS.

Getting Started

In order to see CSS in action, we will need both the HTML file and the CSS file. Download the files linked, and make sure they are in the same folder.

Open your index.html file. If some of the text you see is purple, then you are on the right track.
