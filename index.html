Introduction

CSS is the mechanism that we use to describe the styling of web documents, such as those written in HTML. It is a style sheet language that allows you to pick HTML elements and customize their appearance.

p {
color: purple;
}

The code above turns the text in a paragraph purple. This is one of the many things you can do with CSS. This guide will help you get started on working with CSS.

Getting Started

In order to see CSS in action, we will need both the HTML file and the CSS file. Download the files linked, and make sure they are in the same folder.

Open your index.html file. If some of the text you see is purple, then you are on the right track.

Syntax

A CSS rule consists of a CSS selector and one or more CSS declarations.

p {
color: purple;
}

In the line of code above, the selector is p, which is targeting all paragraph elements. The declaration is color: purple, with color as a property, and purple as the value of the property previously mentioned.

Properties

Let's add another declaration. There are many choices for properties. For example:

  • background-color gives a color to the background of an element
  • border-style specifies the border for an element
  • font-family specifies the font to be used for the text in an element
  • overflow controls what happens when there is too much content to fit in an area
  • opacity specifies how transparent an element is

Let's underline our paragraph text. We will be using the property text-decoration with a value of underline. It should look like this:

p {
color: purple;
text-decoration: underline;
}

If you save index.css and open the index.html file, the purple text should now be underlined.

Basic Selectors

There are different types of CSS Selectors. This page will cover the basic selectors.

Element Selectors

Element selectors allow you to apply CSS to the HTML element named.

p {
color: purple;
}

The code above is an example of an element selector, where all p elements will have purple text.

Here is a reference list of CSS selectors.

The Universal Selector

The universal selector allows you to apply CSS to all HTML elements. Use the asterisk character (*).

* {
text-align: center;
}

The code above affects all HTML elements and makes sure text is center aligned.

Class Selectors

Class selectors allow you to apply CSS to HTML elements that have a matching class name. Use the period character (.) before the class name.

.animal-header {
text-transform: capitalize;
}

The code above is an example of a class selector selector, where all HTML elements with class="animal-header" will have text using only capitalized letters.

Please note that if an HTML element has more than one class, it will use CSS from all of its classes.

Id Selectors

Id selectors allow you to apply CSS to an HTML element with a matching id. Use the hash character (#) before the id name.

#cat-info {
color: pink;
}

The code above is an example of an id selector, where the HTML element with id="cat-info" will have pink text.

Multiple Selectors

If you would like to apply the same CSS to multiple elements but not all, separate the names of the elements with a comma (,).

h1, h2 {
font-style: italic;
}

The code above will make h1 and h2 elements have italic text.
